# Project: uditijeloka.in

## Overview
Personal website hosted on Vercel.
- Repo: https://github.com/uditijeloka/BTC
- Live site: https://uditijeloka.in

## URL Routes
- `/btc` → `index.html`
- `/WIB` → `wib.html`

## Workflow
- All changes are made by Claude and pushed directly to the `main` branch via GitHub API
- Vercel auto-deploys on every push to `main`
- No local development — GitHub is the single source of truth

## Files
- `index.html` — BTC page
- `wib.html` — WIB page
- `vercel.json` — Vercel routing config
- `Resume.pdf` — Resume file
